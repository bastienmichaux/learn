# Java

## Variables

### Data types (primitive types)

```java
boolean x;

char x;

byte x;
short x;
int x;
long x;

float x;
double x;
```

### Constants (`final`)

Variables declared with `final` can be initialized only once.

Prefer constants over variables whenever possible.

Prefer CONSTANT_CASE for identifiers.

```java
final int N_CATS = 4;
```
